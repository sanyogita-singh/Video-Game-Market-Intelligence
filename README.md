# Video-Game-Market-Intelligence
Executive Overview

This project delivers a comprehensive market intelligence analysis of the global video game industry using Python-based analytics and Power BI visualization.

The objective is to identify:

Market-leading publishers

High-performing genres and platforms

Regional revenue concentration

Time-based growth trends

Portfolio and risk exposure

The dashboard enables data-driven strategic insights for investment, publishing strategy, and regional expansion.
